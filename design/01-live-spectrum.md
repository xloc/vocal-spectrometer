# Slice 01: Live Spectrum

## Goal

Capture microphone audio and render a real-time frequency spectrum.

## Behaviour

1. User opens the app and sees a Start button.
2. User clicks Start → browser requests mic permission → real-time frequency spectrum appears on a canvas.
3. User clicks Stop → capture stops.
4. User can click Start again to resume.

## Definition of Done

- Microphone captures audio and spectrum updates in real time
- Stop/start cycle works
